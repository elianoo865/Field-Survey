# ملاحظات مهمة (مطابقة لطلباتك)

## 1) الأدمن يحذف/يضيف أسئلة بأي وقت
- الحذف هنا Soft Delete: يتم وضع isDeleted=true للسؤال.
- السبب: لا نخرب الاستجابات القديمة التي تعتمد على questionId.
- في واجهة الأدمن، السؤال المحذوف سيظهر (محذوف) ويمكن تركه أو إعادة تفعيله عبر تعديل isDeleted.

## 2) Required Questions
- لكل سؤال يوجد switch (Required).
- أثناء تعبئة الاستبيان، أي سؤال Required يمنع الإرسال إذا كان فارغ.

## 3) GPS Mandatory
- زر “إرسال” يرفض الإرسال إذا لم يتم التقاط GPS.
- Firestore Rules أيضاً ترفض إنشاء response بدون location.lat/lng.
